Detailed Description

Gerencia entidades e seus componentes.

O EntityManager armazena e organiza entidades do jogo e seus componentes associados (Posição, Velocidade, Sprite, etc.). Ele fornece métodos para criar novas entidades e recuperar aquelas que possuem componentes específicos.
